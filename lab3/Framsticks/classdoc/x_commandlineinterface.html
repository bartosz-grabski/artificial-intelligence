Command line interface

Command line objects are available in "frams" and "theater".
    Classes defined in Command line interface:
  • ClassBrowser
    Object reference, can be used to provide online hints.
  • CLI
    Commandline support functions.
  • CreatureCharts
    Manages charts for monitoring creatures performance.
  • CreatureSelection
  • DynaChart
    Line chart (visual object).
  • DynaChart2D
    2D Line chart (visual object).
  • GLDisplay
    OpenGL display for the Framsticks Theater application.
  • NeuroDiagram
    Can be used as the client object in the Window.
  • RowFormat
    Used for creating simple listings of objects (in command line applications).
  • ShowManager
  • SimStatsCharts
    Manages charts for monitoring simulator statistics.
  • TrackingCam
    Camera object with the automatic target tracking ability.
  • VisualStyle
  • Window
    Window object, used in the Framsticks Theater application.
